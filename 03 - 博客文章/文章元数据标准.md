# 基本结构

```markdown
---
title: 文章标题
date: 发布日期
categories:
    - 一级分类
    - 二级分类
tags:
    - 标签
math: true              -- 文中存在latex公式时使用
hexo server本地预览时，文章支持热重载，可先校对无误后再部署到github。
---
```
## 文章添加方法：

- 命令`hexo new 文章名`新建文章及附件文件夹
- Obsidian文章内容复制到.md文件，相关图片复制到附件文件夹即可
